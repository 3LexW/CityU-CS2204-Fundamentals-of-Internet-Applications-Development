Style Sheets Explanation:

  1. style.css - Embedded Style
    1. #webpage: Set the font for the entire website
    2. @keyframes logoShift: Animation of shifting the logo from right to it's final position
    3. img[alt="CinemaHK - Logo"]: Applied the animation logoShift
    4. header: The style of the header for all pages
    5. header > h1: Shift to the right for the logo's space
    6. #nav: Set a fixed position to the right of the container, with some transparent to reveal data behind it
    7. #nav > ul: Remove the bullet points
    8. footer: The style of the footer for all pages
    9. footer > *: Spacing between element in the footer
    10. a:hover: Visual guidence of using is hoveing a link
  2. homeStyle.css
    1. a[href="./home.html"]: Disable clicking the home page again
    2. #movieList: Using flex display to create a dynamic view of the webpage
    3. #movieList > div: Padding between elements, and 50% width for two movie lists (Showing and Up coming)
    4. .movie: For each movie, create a border, and apply flex display to adapt difference container size
    5. .movie > img: Uniform the size of the movie poster
    6. .movieName, .cast, .director, .duration: Create spaces between the information of the movie for better visual experience
    7. img[alt *= "thumbnail"]: Make all the movie thumbnails clickable
  3. nowShowing.css
    1. a[href="./nowShowing.html"]: Disable link
    2. #cinemaSelection: Apply flex display to fit the heading and select box on the same row
    3. #cinemaSelection > select: Spacing between two elements on the same line
    4. #movieSelect: Using flex display to prevent a container which has an extreme small size
    5. #movieSelect > h2: Fill the instruction to the entire row under the parent's flex display mode
    6. .movie: Set border and the width for each movie, align all items to the center
    7. .movie > img: Set the size of the poster, and center them to the center of the div
    8. .movieName: Set the alignment of the element to the center of the div
    9. .movie > form: Set the alignment of the element to the center of the div
    10. #cinamaInfor: Create spacing with the previous block, and set to flex mode to adapt different container sizes
    11. #cinemaInfor > h2: Fil lthe instruction to the entire row under the parent's flex display mode
    12. .cinemaDetail: Change the width of the div so that there will be at most three items on a row
    13. .cinemaDetail: Create some spacing between the element and the border of the div
  4. ticketStyle.css
    1. a[href="./ticket.html"]: Disable the click of the link
    2. #movieInformation: Aligned the header and the information into two rows by using flex display
    3. #movieInfomration > h3: Set the field name's width on the webpage
    4. table: Set the width, and spacing with other elements
    5. td: Set cell borders, and align the text
    6. tr:hover : Exclude rows which are not suppose to be selected, and apply background color when the mouse pointer hovers on the cell
    7. input: Style of the input button for better visualization
  5. printStyle.css
    1. .ticket: Use flex display to set two items on each row, also, performed printing style adjustment
    2. .tecket p: Set the p tags to 50%, such that a new field must start a enw row
  6. printDisplay.css
    1. @media print: Imported as print style sheet, and display nonthing for the selected class, therefore some class names are appended in the HTML file "print.html"
